# ⚛️ React 학습 저장소 (React Study Repository)

이 저장소는 리액트(React)를 공부하고 실습한 내용을 일자별로 기록하는 개인 학습 공간입니다.  
기초 문법부터 컴포넌트 설계, 상태 관리 등 매일 학습한 코드와 프로젝트들이 포함되어 있습니다.

## 📅 학습 로드맵 및 폴더 구조

```
├── day1/               # Day 1: React 기초 및 개발 환경 설정
│   ├── vite-proj/      # Vite를 이용한 첫 React 프로젝트 설정
│   └── proj2/          # 기초 컴포넌트 실습
│
├── day2/               # Day 2: ES6+ 문법 및 React 핵심 기초
│   ├── es6.html        # ES6+ 핵심 문법 정리 및 실습
│   ├── proj1/          # JSX, Props, State 기초 실습
│   └── proj2/          # 이벤트 핸들링 및 컴포넌트 매핑 실습
│
└── day3/               # Day 3: 컴포넌트 상태 관리 및 Hook 실습
    └── proj1/          # useState, useEffect 및 컴포넌트 간 데이터 전달 실습
```

## 🛠️ 주요 기술 스택 (Tech Stack)
- **Library:** React
- **Build Tool:** Vite
- **Language:** JavaScript (ES6+)
